# 🗓️ A-Puzzle-A-Day Solver (日历拼图模拟器与求解器)

这是一个基于 Python 和 Pygame 编写的交互式日历拼图游戏与极速自动求解器。无论你是想亲手挑战这款经典的实体解谜玩具，还是想让电脑帮你瞬间找出今天的答案，这个小工具都能完美满足你！

## ✨ 核心亮点

* **丝滑的图形交互**：支持右侧网格缩略图、拖拽瞬间放大、实时旋转与翻转（带有形态记忆功能），以及底板网格吸附。
* **极速回溯求解器**：内置基于 DFS（深度优先搜索）的精确覆盖算法，选好目标日期后按下空格键，毫秒级计算出完美解法！
* **千变万化的答案**：每次按下自动求解，都会随机打乱搜索分支，为你呈现完全不同的拼法组合。
* **轻量级单文件**：所有核心逻辑、算法和 UI 渲染均在一个 `.py` 文件中完成，极致纯粹。

## 🕹️ 操作说明
* **设定目标**：鼠标左键点击底板，选取你要空出来的 月、日、星期（共3个格子，变红即为选中）。

* **手动挑战**：鼠标左键按住右侧的缩略图并拖拽到底板。

* **旋转翻转**：抓起拼图或将鼠标【悬停】在拼图上时，按 R 键旋转，按 F 键翻转。

* **电脑代打**：选好 3 个日期格子后，按 空格键，瞬间自动求解！再次按下可切换另一个解。

* **一键重置**：按 C 键清空底板，拼图全部归位。

* **⚠️ 注意事项**：游玩时请确保关闭中文输入法（切换为纯英文或按 Shift），否则快捷键将失效。
